STEP 1: BACKUP YOUR CURRENT SHEET (IMPORTANT!)
═══════════════════════════════════════════════════════════════

2. Find "SOP_Cal" tab at the bottom
3. RIGHT-CLICK on the tab
4. Select "Duplicate"
5. Rename to "SOP_Cal_Backup"

STEP 2: LOCATE THE CSV FILE
═══════════════════════════════════════════════════════════════

📁 vsm-app/SOP_CAL_COMPLETE_TABLE.csv

1. Go to the "vsm-app" folder
2. Find "SOP_CAL_COMPLETE_TABLE.csv"
3. Remember this location

═══════════════════════════════════════════════════════════════
STEP 3: OPEN IMPORT DIALOG IN GOOGLE SHEETS
═══════════════════════════════════════════════════════════════

1. Open your Google Sheets file
2. Click on "SOP_Cal" tab
3. Click "File" menu (top left)
4. Click "Import"

Menu looks like:
┌─────────────┐
│ File        │
├─────────────┤
│ New         │
│ Open        │
│ Import      │ ← Click here
│ ...         │
└─────────────┘

═══════════════════════════════════════════════════════════════
STEP 4: UPLOAD THE CSV FILE
═══════════════════════════════════════════════════════════════

Import dialog appears with 3 tabs:
┌──────────────────────────────────────┐
│ Upload | My Drive | Shared with me   │
└──────────────────────────────────────┘

1. Click "Upload" tab
2. Click "Browse" button (or drag & drop)
3. Navigate to: vsm-app folder
4. Select: SOP_CAL_COMPLETE_TABLE.csv

File will upload (takes 1-2 seconds)

═══════════════════════════════════════════════════════════════
ALTERNATIVE METHOD: COPY-PASTE (IF IMPORT DOESN'T WORK)
═══════════════════════════════════════════════════════════════

If File → Import doesn't work:

METHOD A: Open CSV in Excel/Notepad
1. Right-click SOP_CAL_COMPLETE_TABLE.csv
2. Open with Notepad (or Excel)
3. Select all (Ctrl+A)
4. Copy (Ctrl+C)
5. Go to Google Sheets
6. Click cell A1 in SOP_Cal sheet
7. Paste (Ctrl+V)
8. Google Sheets will parse the CSV automatically

METHOD B: Direct Paste
1. Open SOP_CAL_COMPLETE_TABLE.csv in your code editor
2. Copy all content
3. Go to Google Sheets
4. Click cell A1
5. Paste
6. Done!

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Can't find "Import" in File menu
Solution: Make sure you're in Google Sheets (not Google Docs)

Problem: Import button is grayed out
Solution: Make sure you selected "Replace current sheet"

Problem: Data looks wrong after import
Solution: Try copy-paste method instead

Problem: Columns in wrong order
Solution: Your original sheet might have different column order
         Check column headers and rearrange if needed
