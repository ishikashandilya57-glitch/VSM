STEP 1: Open the CSV File (1 minute)

1. On your computer, open: vsm-app\SOP_CAL_CORRECT_DATA.csv
2. Open it with Notepad or Excel
3. Select ALL (Ctrl+A)
4. Copy (Ctrl+C)

═══════════════════════════════════════════════════════════════════

STEP 2: Paste into Google Sheet (1 minute)

1. Open your Google Sheet
2. Go to SOP_Cal tab
3. Find an empty area (or create a new temporary sheet)
4. Click on a cell
5. Paste (Ctrl+V)
6. The data will appear in columns

═══════════════════════════════════════════════════════════════════

Now you have 25 rows with correct structure:
- 5 Sewing rows (Q1-Q5)

- 5 Finishing rows (Q1-Q5)

For each row:
1. Check if it already exists in your SOP_Cal
2. If exists → Replace the old row with the new one
3. If doesn't exist → Add the new row

KEY: Make sure columns match:
- Column A: Process Seq
- Column B: Process Stage
- Column C: Product Type (must be "All")
- Column D: Wash Category
- Column E: Order Type (must be "All")
