🎯 WHAT CHANGED - PRODUCT TYPE FEATURE
========================================

## BEFORE (Old Behavior)
------------------------
❌ Product Type dropdown visible for ALL processes
❌ User had to select Product Type every time
❌ Risk of selecting different Product Type for same OC
❌ Confusing for users


## AFTER (New Behavior)
-----------------------
✅ Product Type dropdown ONLY visible for Pre-Production
✅ User selects Product Type ONCE (in Pre-Production)
✅ All subsequent processes automatically use that Product Type
✅ Clean, simple, user-friendly

## HOW IT WORKS
---------------

STEP 1: Pre-Production
  User: Selects "Pre-Production"
  UI: Shows Product Type dropdown
  User: Selects "Shirt"
  System: Saves "Shirt" to Column AK in Order_Master

STEP 2: Cutting
  User: Selects "Cutting"
  UI: Product Type dropdown HIDDEN
  System: Retrieves "Shirt" from Order_Master
  System: Uses "Shirt" for Cutting SOP calculation
  Result: Cutting (Shirt, Q1) = 1.9 days ✅

STEP 3: Sewing
  User: Selects "Sewing"
  UI: Product Type dropdown HIDDEN
  System: Retrieves "Shirt" from Order_Master
  System: Uses "Shirt" for Sewing SOP calculation
  Result: Sewing (Shirt, Q1) = 1.7 days ✅

... and so on for all processes!
