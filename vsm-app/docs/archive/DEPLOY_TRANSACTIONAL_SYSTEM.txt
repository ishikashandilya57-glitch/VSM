❓ TROUBLESHOOTING:

Problem: Formulas not appearing
Solution: Check Apps Script logs (View → Logs)
         Look for: "✅ Formulas added for transactional process"

Problem: Cumulative not calculating correctly
Solution: Check that OC_NO and PROCESS_STAGE match exactly
         Formula is case-sensitive

Problem: UI not showing current progress
Solution: Make sure Apps Script is deployed
         Check browser console (F12) for errors
